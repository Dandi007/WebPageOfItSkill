Linux Shell环境手动安装Eclipse

[日期：2014-11-12] 来源：Linux社区  作者：kleguan [字体：大 中 小]

PC环境：CentOS 6.5

Eclipse版本： eclipse-jee-kepler-SR2-linux-gtk-x86_64.tar.gz

（下载地址; http://www.eclipse.org/downloads/download.php?file=/technology/epp/downloads/release/kepler/SR2/eclipse-standard-kepler-SR2-linux-gtk-x86_64.tar.gz） 

假定已安装完成JDK，下面介绍安装步骤。

1. 利用su切换到root模式或在以下每个命令前加上sudo.

2. 解压eclipse

    tar -xvzf eclipse-jee-kepler-SR2-linux-gtk-x86_64.tar.gz-C /opt

    说明：（1）待解压文件名根据下载版本变化；（2）利用 -C 将目标地址切换为  /opt ， 目标地址可自由变换，但推荐为/opt

3. 更换当前路径

    cd /opt

4. 更改eclipse相关权限

    chmod -R +r /opt/eclipse

    touch /usr/bin/eclipse

    chmod 755 /usr/bin/eclipse

5. 添加从shell启动的脚本

    vi /usr/bin/eclipse

将下面内容写入到文件中并保存

  #!/bin/sh

  export ECLIPSE_HOME="/opt/eclipse" 

  $ECLIPSE_HOME/eclipse $*

6. 配置结束，可在shell端输入eclipse命令启动程序，并设定工作空间。
